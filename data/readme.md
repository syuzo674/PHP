# フィードバックフォーム

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - PHPを用いてフィードバックが登録できるWebアプリ

  - ユーザ名とメールアドレス、年齢、フィードバックの内容がユーザ名のファイルとして登録される。<br>ファイルの確認はユーザ名を指定することで任意のファイルが見れるようにしている。

## 工夫した点

  - 任意のファイルをファイル名を指定することで確認できるようにした点

## 苦戦した点

  - 現状、ファイル名の指定で見れるのみとなっているため、ファイル名の指定箇所を認証形式にしてユーザ名とパスワードが一致した人しか見れないようにしたい。
    - 最終的には、登録したファイル名は、登録した本人と管理者権限(admin,root)をつけた人しか見れないように実装予定
